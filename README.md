# C++ OOP Advanced Tasks

Репозиторий содержит решение трех задач по C++: генерация чисел, маскирование контейнеров и менеджер памяти.

## 1. SimpleRNG
Класс для генерации псевдослучайных чисел методом линейной конгруэнтности:
$X_{N+1} = (a \cdot X_N + c) \pmod m$.
- В качестве типа данных используется `double`, остаток от деления вычисляется через `std::fmod`.
- Реализован Input Iterator.
- Метод `end(eps)` создает итератор-стражник. Сравнение с ним возвращает `true`, когда текущее значение генератора приближается к начальному (`_start_val`) ближе чем на `eps`. Это сигнализирует о зацикливании генератора.

## 2. Mask
Шаблонный класс `Mask<N>`, хранящий паттерн из нулей и единиц.
- Конструктор проверяет количество аргументов (`sizeof...(Args) == N`) и их значения (только 0 или 1).
- `slice(container)`: Проходит по контейнеру и перезаписывает элементы "на месте", сдвигая нужные в начало. Лишние элементы в конце удаляются через `resize`.
- `transform` и `slice_and_transform`: Генерируют новые контейнеры, применяя функтор к элементам, на которые выпадает `1` в маске.

## 3. MemReserver
Класс для управления статическим буфером памяти.
- Память выделена как массив `std::byte` с выравниванием `alignas(T)`.
- **create:** Конструирует объект через Placement New.
- **del:** Явно вызывает деструктор `~T()`. (В коде метод назван `del` или `delete_obj`, так как `delete` является ключевым словом).
- **position:** Вычисляет индекс объекта через разность указателей `(addr - start) / sizeof(T)`. Включает проверку границ и выравнивания.
