# Домашка по C++ (ООП)

Решения трех задач. Код компилируется на C++17.

## 1. SimpleRNG
Генератор псевдослучайных чисел.
*   Сделал через формулу `(a * X + c) % m`.
*   Так как числа `double`, использовал `std::fmod` вместо обычного процента.
*   Прикрутил итератор (InputIterator), чтобы можно было юзать в циклах.
*   Самое хитрое: метод `end(eps)`. Он возвращает итератор, который проверяет, не зациклился ли генератор (сравнивает текущее число с начальным с учетом погрешности `eps`).

## 2. Mask
Класс маски.
*   Размер маски задается в шаблоне `<N>`.
*   В конструкторе проверяю количество аргументов через `static_assert`, чтобы сразу била ошибка компиляции, если что не так.
*   `slice`: проходит по контейнеру и сдвигает элементы, которые попадают под единицу в маске, в начало. Потом делает `resize`.
*   `transform`: создает копию и меняет элементы по маске.
*   Маска работает циклично (индекс % N).

## 3. MemReserver
Свой менеджер памяти (статический).
*   Память выделяю как массив байт `std::byte`. Обязательно использую `alignas(T)`, чтобы выравнивание было правильным.
*   `create`: ищет свободный слот (массив `bool`) и вызывает `placement new`.
*   `delete`: руками вызывает деструктор объекта `~T()` и освобождает слот.
*   `position`: считает смещение указателя от начала массива и делит на размер типа. Проверяет, что указатель реально указывает на наш объект.
